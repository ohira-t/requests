# Requests クイックスタートガイド

**5分で始めるタスク管理**

---

## 📱 初めてのログイン

1. ブラウザで Requests にアクセス
2. **「新規登録」** をタップ
3. 名前、メールアドレス、パスワードを入力
4. **「登録」** をタップ

✅ サンプルタスクが3つ作成されるので、触ってみましょう！

---

## 🎯 基本の3ステップ

### ステップ1: タスクを作る

```
右下の「＋」ボタン
  ↓
タイトルを入力
  ↓
担当者を選ぶ（自分でOK）
  ↓
「保存」
```

### ステップ2: タスクを確認する

```
タスクをタップ
  ↓
詳細が表示される
  ↓
必要なら編集
```

### ステップ3: タスクを完了する

```
チェックボックスをタップ
  ↓
「完了する」を選択
  ↓
完了！
```

## 🌟 覚えておきたい機能

### 画面下部のメニュー

| アイコン | 機能 |
|---------|------|
| 📋 課題 | タスク一覧（メイン画面） |
| 📅 カレンダー | 期限を月単位で確認 |
| 📊 統計 | タスクの状況を可視化 |
| ⚙️ 設定 | カテゴリーやユーザー管理 |

### 優先度の使い分け

| 優先度 | いつ使う？ |
|--------|-----------|
| 🔴 緊急 | 今日中に終わらせるべきもの |
| 🟠 高 | 今週中に対応すべきもの |
| 🔵 中 | 通常のタスク（デフォルト） |
| ⚪ 低 | 時間があるときにやるもの |

### タブの使い分け

| タブ | 使うとき |
|------|---------|
| 自分の課題 | 自分がやるべきタスクを確認 |
| 依頼した課題 | 他の人に頼んだタスクの進捗確認 |
| クライアント別 | お客様関連のタスク管理 |

## 💡 便利な使い方

### ピン留め機能

よく使う担当者を **スター** でピン留めすると、リストの上に表示されます。

```
タスク作成画面
  ↓
担当者を選択
  ↓
よく使う人の右側の★をタップ
  ↓
次回から上に表示される
```

### カテゴリーで整理

タスクをプロジェクトやジャンルで分類できます。

```
設定⚙️
  ↓
カテゴリー
  ↓
「＋カテゴリーを追加」
  ↓
名前と色を設定
```

### コメントで会話

タスクについて質問したり、進捗を報告できます。

```
タスクをタップ
  ↓
下にスクロール
  ↓
コメント欄に入力
  ↓
送信
```

## ⚠️ よくある質問

### Q: 間違えてタスクを完了してしまった！
**A**: 「完了」タブからタスクをタップして「未完了に戻す」を選択してください。

### Q: 期限を過ぎたタスクはどうなる？
**A**: 赤色で表示され、「期限切れ」タブに自動的に振り分けられます。

### Q: スマホでもPCでも使える？
**A**: はい、どのデバイスでも同じデータを共有できます。

### Q: 通知が来ない
**A**: 画面右上のベルアイコンをタップして、通知一覧を確認してください。

### Q: カテゴリーの色を変更したい
**A**: 設定⚙️ → カテゴリー → 編集ボタン（ペンアイコン）→ 色を選択

## 🚀 次のステップ

基本操作に慣れたら、こんな使い方も試してみましょう：

1. **チームで使う**
   - スタッフを追加して、タスクを割り当ててみる
   - コメントでやり取りする

2. **カレンダーを活用**
   - 今月のタスクを一覧で確認
   - 期限が近いものを優先的に片付ける

3. **統計で振り返る**
   - どれくらいタスクを完了したか確認
   - 期限切れが多い場合は見直す

4. **カテゴリーで整理**
   - プロジェクトごとに色分け
   - 自分のタスクを視覚的に管理
