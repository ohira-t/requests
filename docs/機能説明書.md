# Requests - 機能説明書

**バージョン**: 1.0  
**最終更新**: 2026年1月

---

## 📝 このアプリについて

**Requests（リクエスト）** は、社内のタスク管理とクライアントからの依頼を一元管理するWebアプリケーションです。

### できること

- ✅ 自分のタスクを管理
- 📤 スタッフ・クライアントへタスクを依頼
- 📊 タスクの進捗状況を可視化
- 📅 カレンダーで期限を確認
- 📱 スマートフォン・タブレット・PCで利用可能

---

## 👥 利用者の種類

### 1. 管理者（Admin）
- **できること**: 全ての機能が使える
  - ユーザー管理（追加・編集・削除）
  - 部署・カテゴリーの管理
  - お知らせの投稿
  - 全タスクの閲覧・編集
  - アカウントの無効化

### 2. スタッフ（Staff）
- **できること**: 
  - 自分のタスク管理
  - 他のスタッフ・クライアントへの依頼
  - 依頼されたタスクの確認・完了
  - クライアントタスクの管理
  - カテゴリー・部署の閲覧

### 3. クライアント（Client）
- **できること**:
  - 自分が依頼したタスクの閲覧
  - タスクの進捗確認
  - コメントでのやり取り

## 🎯 主な機能

### 1. タスク管理

#### タスクの作成
- タイトル、説明文を入力
- 担当者を選択（自分/スタッフ/クライアント）
- 優先度を設定（緊急・高・中・低）
- 期限を設定
- カテゴリーで分類

#### タスクの表示

**3つのタブで整理:**

| タブ | 表示内容 | 対象者 |
|------|---------|--------|
| 自分の課題 | 自分が担当のタスク | スタッフ |
| 依頼した課題 | 社内スタッフに依頼したタスク | スタッフ |
| クライアント別 | クライアントに関連するタスク | スタッフ |

#### タスクの状態

```
未着手 → 進行中 → 完了
```

#### 優先度

| 優先度 | 色 | 用途 |
|--------|-----|------|
| 🔴 緊急 | 赤 | 最優先で対応が必要 |
| 🟠 高 | オレンジ | 早めの対応が必要 |
| 🔵 中 | 青 | 通常の優先度 |
| ⚪ 低 | グレー | 時間があるときに対応 |

### 2. カレンダー表示

- 月単位でタスクの期限を確認
- 期限が近いタスクを色分け表示
  - 自分のタスク: 青
  - 依頼したタスク: オレンジ
  - 期限切れ: 赤

### 3. 統計（ダッシュボード）

**3つのカテゴリーで状況を把握:**

1. **自分の課題**
   - 進行中のタスク数
   - 完了したタスク数
   - 期限切れのタスク数
   - 完了率

2. **依頼した課題**
   - 依頼中のタスク数
   - 完了したタスク数
   - 期限切れのタスク数
   - 完了率

3. **クライアント別**
   - クライアント関連のタスク数
   - 完了状況
   - 完了率

### 4. 設定（管理者・スタッフ）

#### カテゴリー管理
- タスクを分類するカテゴリーを作成
- 色分けで視覚的に管理
- 並び替えで優先順位を設定

#### 部署管理
- 社内の部署を登録
- スタッフを部署に所属させる
- 色分けで識別

#### ユーザー管理
- スタッフ・クライアントの追加
- 権限（管理者/スタッフ）の設定
- よく使うユーザーをピン留め
- 並び替えで優先順位を設定

### 5. 通知機能

- 新しいタスクが割り当てられた
- タスクにコメントがついた
- タスクが完了した
- お知らせが投稿された

### 6. コメント機能

- タスクごとに会話
- 進捗状況の共有
- 質問・回答のやり取り

### 7. データエクスポート

- タスク一覧をCSVでダウンロード
- Excelで開いて分析・管理可能

## 💡 使い方の流れ

### 1. 新しいタスクを作成する

1. 右下の **「＋」ボタン** をタップ
2. タイトルを入力
3. 担当者を選択（自分 or 他のスタッフ or クライアント）
4. 優先度と期限を設定
5. **「保存」** をタップ

### 2. タスクを完了する

1. タスクの左側の **チェックボックス** をタップ
2. 完了確認ダイアログで **「完了する」** を選択

### 3. タスクの詳細を見る・編集する

1. タスクカードをタップ
2. 詳細画面で内容を確認・編集
3. **「保存」** をタップ

### 4. コメントを追加する

1. タスク詳細画面を開く
2. 下部の **コメント欄** に入力
3. **送信** をタップ

### 5. カレンダーで期限を確認する

1. 下部メニューの **「📅」** をタップ
2. 月単位でタスクを確認
3. 日付をタップすると、その日のタスク一覧を表示

## 📞 サポート

### お問い合わせ
- システム管理者にご連絡ください

### よくある質問

**Q: パスワードを忘れました**  
A: 管理者に連絡して、パスワードをリセットしてもらってください。

**Q: タスクが削除できません**  
A: 削除機能は現在実装されていません。ステータスを「完了」にすることで非表示にできます。

**Q: スマートフォンでも使えますか？**  
A: はい、スマートフォン・タブレット・PCすべてで快適に使えるよう設計されています。

**Q: 通知はどこで確認できますか？**  
A: 画面右上のベルアイコンをタップすると、通知一覧が表示されます。
